# Melhoria do Modo Escuro no Dashboard

## Objetivo
Refinar a adaptação visual do Dashboard para garantir consistência e legibilidade no Modo Escuro (Dark Mode).

## Alterações
- Atualizadas cores de fundo e texto da `SidebarItem` para melhor contraste em dark mode.
- Ajustadas cores de hover na sidebar para utilizar transparências sutis (`whiteAlpha.100`).
- Atualizadas cores de texto da Navbar (nome do usuário e e-mail) para se adaptarem ao tema escuro.
- Garantida a consistência das bordas entre Navbar, Sidebar e Área de Conteúdo.

## Regras de negócio
- Toda a interface administrativa deve ser 100% compatível com o tema escuro.
- A experiência visual deve ser premium e confortável para o usuário.

## Observações
- Utilizado `blue.200` para itens ativos em dark mode para maior suavidade.
- Fundo do dashboard em dark mode utiliza `gray.950` para profundidade.
